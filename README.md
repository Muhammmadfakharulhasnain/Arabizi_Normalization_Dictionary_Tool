# Arabizi Normalization Dictionary Tool

A web-based tool for normalizing Arabizi words and their variations using React (frontend) and Express.js (backend).

## 🔧 Features
- Search for Arabizi words and view normalized variations
- Add new variations to existing keys
- Create new keys with multiple variations
- Stores data in a local JSON file

## 🛠️ Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Data: JSON dictionary
